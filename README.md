# Colin Minor Author Website

Static author website for Colin Minor and `Hitler's Spy: A Novel of Deception`.

## Local preview

```bash
python -m http.server 4173
```

Open `http://127.0.0.1:4173/`.

## Structure

- `index.html` - semantic page markup and copy
- `styles.css` - responsive premium historical-fiction styling
- `script.js` - mobile navigation and footer year behavior
- `assets/author/` - author portrait
- `assets/books/` - book cover image
- `assets/book-displays/` - book display images
